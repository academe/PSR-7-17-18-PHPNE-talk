* Application wants to send a request to a service. * The service retrieves some information. * The application gets that information returned in a response. * A HTTP client orchestrates that.
* So, you write a package that sends requests and receives responses. * The package has a client injected. * It now needs to create PSR-7 requests, * A factory can do that, and such a factory is a PSR - PSR-17.
* These are interfaces * Message is not used directly; it provides common methods * Request is constructed by your app/package to send * Request is constructed by the client on receipt * ServerRequest is constructed by the framework on inbound request
